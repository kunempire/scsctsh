.TH MYLS 1 "November 2023" "Version 1.0" "User Commands"

.SH NAME
myls \- list directory contents.

.SH SYNOPSIS
.B myls
[OPTION] [FILE]

.SH DESCRIPTION
List information about the FILE (the current directory by default) and sort entries alphabetically.

.SH OPTIONS
.B \-l
: use a long listing format with details.
