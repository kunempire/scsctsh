.SH NAME
myps \- report a snapshot of the current processes.

.SH SYNOPSIS
.B myps
[-p pid]

.SH DESCRIPTION
Myps displays information about the active processes, all by default.

.SH OPTIONS
.B \-p 
\fI[pid]\fP: display the information of processe \fIpid\fP.
